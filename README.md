## Development

### Run a presentation locally

```bash
# Install dependencies
cd allure-reporting
npm install

# Start dev server
npm run dev
```

### Add a new presentation

1. Create a new folder: `mkdir my-new-talk`
2. Initialize Slidev: `cd my-new-talk && npm init slidev`
3. Update `.github/workflows/deploy.yml` to include the new presentation
4. Update `index.html` to add a card for the new presentation
